# Installation instructions 
### Arch Linux
```
 git clone https://aur.archlinux.org/posy-cursors.git
 cd posy-cursors
 makepkg -si
```
<sub><sup>yay -S posy-cursors</sup></sub>

### Manual

Run `bash create.sh`

Copy `posy-*` folders to `~/.icons`
